## Step 3: Create Your First Booking (2 minutes)

### **Option A: Create New Request**

1. **Navigate to Requests**
   - Click "Requests" in the sidebar
   - Click "Create New Request" button

2. **Enter Customer Information**
   - Fill in required fields:
     - First Name *
     - Last Name *
     - Date of Birth
     - Nationality
     - Passport Number
   - Click "Save"

3. **Add Booking Details**
   - Go to "Main Table" section
   - Find your new request
   - Click on any cell to edit inline
   - Enter:
     - Airlines Price
     - Service Ticket
     - Visa details (if applicable)
     - Payment information

4. **Save Automatically**
   - Changes save automatically when you click away
   - Orange highlight shows "Total Amount Due"
   - Green highlight shows "Payment Balance" when fully paid

**âœ… Your first booking is created!**

## Common Questions

**Q: How do I edit a booking?**  
A: Click any cell in the Main Table to edit inline. Changes save automatically.

**Q: Where do I see payment status?**  
A: Check the "Payment Balance" column. Green = fully paid, Red = customer owes.

**Q: How are totals calculated?**  
A: All calculations are automatic. See [Financial Calculations](../02-financial/calculations.md) for details.

**Q: Can I undo changes?**  
A: Changes save automatically. Contact your administrator if you need to correct data.
